Break into tech with the needed skill set.

Motivated by the need for emerging developers and designers to create lasting solutions to global challenges, the Zuri & Ingressive 4 Good internship is a 4 months remote training where interns are exposed to the tech world. The interns select their choice field and get to learn from professionals who put them through the basics via tutorials, live projects and career mentorship.

Track

The internship will cover the following areas

Design
UI/UX

This track deals with creating interface for websites or mobile applications. Using Figma, interns will be taught on how to convert project documentation into viewable and understandable graphic design for the developers to work with.

Front-End Web Development
HTML | CSS | Javascript

This track deals with coding the part of the website or mobile application that users interact with. The front-end developer intern will be taught on how to correctly interpret a given user interface design to reproduce a perfect application.

Back-End Web Development
PHP | Python | JavaScript

This track deals with the aspect of the website application the user does not directly interact with. It allows the frontend function either by running some back-end code or connecting to the database.

Mobile Development
Java | Flutter | Kotlin

The mobile development track contains of a a list of tools and languages needed for a developer to create mobile applications. The mobile application interns will be taught to develop, are either IOS or Android apps.
